## Skill catalog

- `manage-workspace-agents-md`: create, patch, audit, and align repository
  `AGENTS.md` guidance from evidence.
- `pre-1-0-forward-only`: keep pre-1.0 Rust APIs and durable documentation
  focused on the current repository shape.
- `rust-best-practices`: write, refactor, review, optimize, and document
  idiomatic Rust on the tracked stable baseline.
- `rust-test`: design, patch, audit, and validate Rust tests across unit,
  integration, doctest, diagnostic, snapshot, async, unsafe, fuzz, feature,
  target, MSRV, coverage, and benchmark surfaces.
